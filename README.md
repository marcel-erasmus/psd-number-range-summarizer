# ABOUT

Demo project to summarize a range of integers.

# EXAMPLES

### EXAMPLE 1

**INPUT**
```
1, 2, 3, 4, 5, 10
```

**OUTPUT**
```
1-5, 10
```

### EXAMPLE 2

**INPUT**
```
-1, -2, -3, -4, 10, 11, 12, 13, 14, 19
```

**OUTPUT**
```
-4--1, 10-14, 19
```

### EXAMPLE 3

**INPUT**
```
5, 5, 5, 5, 5, 5, 5, 6, 3, 3, 9
```

**OUTPUT**
```
3, 5-6, 9
```
